<h2>프로젝트 소회</h2>
<a id="프로젝트-소회" class="anchor" aria-hidden="true" href="#프로젝트-소회"></a>
<p>한달여간 프로젝트를 진행하며 단시간에 많은 것을 머릿속에 넣다보니 때로는 과부하가 오기도 하고 포기하고 싶은 순간도 많았습니다. 
   그렇지만 성실한 팀원들을 만나 마침내 프로젝트 마침표를 찍을 수 있음에 감사합니다. <br>
   이용자 입장에서 웹사이트를 이용했을때 별 거 아니라고 생각했던 기능들도 많은 로직이 필요하구나 새삼 느끼게 되었습니다.<br>
   팀원들끼리 각자 작업하다가 팀원 중 한 명이 총대를 매고 프로젝트를 합쳐 깃에 배포하는 형식으로 프로젝트를 진행하였는데,
   그러다보니 버그도 많이 발생하고 코드끼리 겹치는 경우가 많아 더더욱 깃의 중요성을 깨닫게 되었습니다.<br>
   또한 개인적으로는 페이징 처리를 하는데 애를 많이 먹었고, 
   초심자이다보니 동일한 기능은 xml쪽에서 단일 쿼리문으로 if만 돌려줘도 됐을텐데 괜히 controller에 중복되는 기능을 넣어 
   좀 더 가독성 좋게 코드를 만들지 못한 것 같아 아쉬움이 많이 남습니다.<br>
   이번 프로젝트의 아쉬움을 발판 삼아 다음 프로젝트는 레스트풀하게 코드를 짜는 것이 목표입니다.
</p>
