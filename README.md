<h2>프로젝트 소회</h2>
<a id="프로젝트-소회" class="anchor" aria-hidden="true" href="#프로젝트-소회"></a>
<p>한달간의 프로젝트 진행 동안 여러 가지 어려움을 겪었지만, 성실한 팀원들을 만나 마침내 프로젝트를 완성할 수 있어서 감사합니다. 웹사이트를 이용자로서 사용하면서 기능들이 간단해 보였던 것들도 실제로는 많은 로직이 필요한 것을 깨달았습니다.

팀원들끼리 각자 작업하다가 한 명이 총괄하여 프로젝트를 깃에 배포하는 방식으로 진행했던 결과, 버그가 많이 발생하고 코드의 중복이 많아졌다는 점에서 깃의 중요성을 더욱 깨닫게 되었습니다. 이러한 경험으로부터 깃의 중요성을 인지하게 되었습니다.

또한 개인적으로는 페이징 처리에 많은 어려움을 겪었고, 초심자로서 가독성을 떨어뜨리는 실수를 저지른 것 같아 아쉬움이 많이 남습니다. 이번 프로젝트에서의 아쉬움을 바탕으로 다음 프로젝트에서는 더욱 레스트풀한 코드를 작성하는 것을 목표로 삼고자 합니다.
</p>
